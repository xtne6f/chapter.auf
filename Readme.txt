■概要
Fork元「チャプター編集 for AviUtl by ぽむ + 無音＆シーンチェンジ検索機能 by ru」
にYUY2フィルタモードでの動作安定など細かい修正と、いくらかの余計な機能を追加した
ものです。カット編集とチャプター編集の両方での利用を想定しています。

chapter_exeのほうはまったく弄っていません。chapter.aufに対する改変です。

# PullRequestしたいけどゴテゴテ機能追加しすぎかもしれない…わりと自分用

■開発環境
・Windows8(x64)
・Microsoft Visual C++ 2010 Express SP1
・Aviutl 1.00

■上級者向け情報
ダイアログで設定可能な項目以外に、aviutl.iniの"[チャプター編集]"セクションに以下
のキーを追記してカスタマイズすることができます。
◇hideList
  非表示にしたいボタンやチェックボックスを指定
  SV=保存   LD=読込   DM=無音部分 EM=無音削除 XE=編集点展開 MS=連続
  MT=閾値   SC=SC位置 SA=全SC検出 SM=mark付与 AU=自動出力   DS=既定保存
  FX=拡張子 TH=サムネ TL=ﾀｲﾑﾗｲﾝ   TT=総時間
  上記のリストを参考に、例えば編集点展開ボタンと拡張子エディットボックスを非表示
  にしたいときは hideList=XE,FX のように指定します。
◇editStartMark / editEndMark
  編集点展開後の編集点開始/編集点終了チャプターの名称を指定
