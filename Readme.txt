■概要
Fork元「チャプター編集 for AviUtl by ぽむ + 無音＆シーンチェンジ検索機能 by ru」
にYUY2フィルタモードでの動作安定など細かい修正と、いくらかの余計な機能を追加した
ものです。カット編集とチャプター編集の両方での利用を想定しています。

chapter_exeのほうはまったく弄っていません。chapter.aufに対する改変です。

# PullRequestしたいけどゴテゴテ機能追加しすぎかもしれない…わりと自分用

■開発環境
・Windows8(x64)
・Microsoft Visual C++ 2010 Express SP1
・Aviutl 0.99m
